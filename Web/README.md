#### ⚡ 使用说明

建议使用 pnpm，因为 yarn 有时会报错。<a href="http://nodejs.cn/" target="_blank">node 版本 > 14.18+/16+</a>


```bash
# 安装依赖
pnpm install

# 运行项目
pnpm run dev

# 打包发布
pnpm run build
```
